abstract suspend fun storeMessage(connectionId: String, cipherText: String): RelayMessage
Content copied to clipboard

Stores a message with the text contents of cipherText in the relay postbox with ID of connectionId. The message stored in the postbox will have a timestamp of the current system time, a random v4 UUID string as messageID, and a direction of OUTBOUND.

Return

the RelayMessage that was stored in the postbox.

Parameters

connectionId

the postbox identifier to store a message in.

cipherText

the string of text to store.
