abstract fun connectionStatusChanged(state: DIRelayEventSubscriber.ConnectionState)
Content copied to clipboard

Notifies the subscriber that the subscription connection state has changed. The subscriber won't be notified of RelayMessage changes until the connection status changes to ConnectionState.CONNECTED. The subscriber will stop receiving notifications when the connection state changes to ConnectionState.DISCONNECTED.
